---
title: "Extract Paystub PDF API"
description: "Extracts paystub information from a PDF file."
api: "POST /extract-paystub-pdf"
---

<Note>
### PDF Extraction Only

This endpoint processes documents in PDF format. Please convert to PDF before
submitting an API request.

</Note>

### Headers

<ParamField body="x-api-key" required>
  Your API key.
</ParamField>

### Body

<ParamField body="file" required>
  The PDF file to extract paystub information from.
</ParamField>

### Example Request

```bash
curl --location 'https://2y31yfw1hd.execute-api.us-east-1.amazonaws.com/prod/extract-paystub-pdf' \
--header 'x-api-key: {{api_key}}' \
--form 'file=@"/Users/shishirkumar/Downloads/pay-stub-template-classic.pdf"'
```
